Help & Support

Guides and common fixes

Get help

Start with the app documentation for features and how-tos. For store-related billing issues, use Google Play’s subscription / order help for StreakForge (com.bashtect.streakforge).

Common questions

Where is my data stored?
On your device only. Use Settings → Backup Database to create a backup file.

Privacy / Terms links fail?
Open them from Settings after updating the app, or visit this site directly: Privacy · Terms.

Reminders not firing?
Allow notifications, allow alarms/reminders, and disable battery restrictions for StreakForge (especially on MIUI).

Notification +1 / Snooze / Skip?
Tap the action on the reminder; the app applies it when it opens. Snooze reschedules in about 10 minutes.

Catalog not loading?
Check internet for the first download. Cached templates work offline afterward.
